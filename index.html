Gink

Gink is a versioned, eventually consistent, multi-paradigm database management system. It takes a "protocol-first" approach, which facilitates multiple implementations that can share data. My personal contributions to Gink have included implementing new data structures, such as Role, PairMap, PairSet, KeySet, Box, and more, as well as creating a MemoryStore which caches the database in-memory using Tree Maps in TypeScript in order to improve read performance compared to IndexedDb. I have also handled most of the development operations, primarily integration testing in the browser, documentation generation and deployment, and automatic package publishing.

JavaScript/TypeScript, Python, Docker, Linux (Debian/Ubuntu), Make, protocol buffers, Webpack, GitHub Actions (unit/integration testing, documentation generation, linting, and package publishing to PyPI and NPM).

Bet

I created a webapp to allow users to "bet" on stocks, long or short. However, there is no real money involved. Each bet uses a "chip", and each user begins with one chip. The site allows users to track their stats for correct and incorrect bets, and even place stops to close a bet based on the current stock price. Bet uses the Polygon API for real-time stock data. Through this project, I have gained experience hosting, deploying Docker containers, and running scheduled jobs through Google Cloud and cron.

JavaScript, Neo4j Graph Database, CYPHER Query Language, Python, Flask, Docker, Google Cloud Run, Google Cloud Scheduler, HTML, CSS

Various APIs

Website Screenshot: A simple API with only one endpoint (for now), which, upon GET request, responds with an image of the website at the provided URL. This API combines Express and Puppeteer to quickly start a headless browser and take a screenshot of any website.

Tons of Words: Access over 370,000 English words in many different ways, whether they are random, filtered, or indexed. Created using JavaScript and Express, and hosted on Heroku.

JavaScript, Express, Puppeteer, Heroku

Typing

Typing is a simple web application designed to help users improve at typing and spelling in a language they are learning. Currently, English and Spanish are the languages offered. Once the user selects their language and difficulty, they will hear a word spoken by speechSynthesis API, and they will need to type out the word and spell it correctly. This project uses vanilla JavaScript and web components for the frontend, and Express for a backend.

JavaScript, Express, spacey (to clean words list)
